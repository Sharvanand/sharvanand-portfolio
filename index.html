Weather Application

view online code on github

App was designed to check the city weather. The openweather map public API was used to fetch the data. The data flows from API so there is no backend involved in this app. React, Redux and Materialize Css were used to build the app.

The redux was used to handle the state management. Async/Wait Fucntion was used in action creators to call the api. i used Async/Wait function to handle the delay response from API.
